# Student Management System

## Description
A full-stack web application to manage student records.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- MongoDB Database Integration

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Run Project

1. Install dependencies
   npm install

2. Create .env file

3. Start server
   node server.js

4. Open
   http://localhost:5000
